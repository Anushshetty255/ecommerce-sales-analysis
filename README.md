# ecommerce-sales-analysis
This project analyzes e-commerce sales data using SQL and Power BI. 
It focuses on identifying sales trends, growth patterns (YoY, MoM), and performance across categories, regions, and products.

Tools Used
- SQL (Data Analysis)
- Power BI (Dashboard & Visualization)
- Excel / CSV (Data Source)

  Key Analysis
- Total Sales Calculation  
- Monthly Sales Trend Analysis  
- Year-on-Year (YoY) Growth  
- Month-on-Month (MoM) Growth  
- Category-wise Performance  
- Region-wise Sales Analysis  
- Top Products Identification


Key Insights
- Electronics is the highest revenue-generating category  
- Sales fluctuate across months indicating inconsistent demand  
- West region contributes the highest revenue  
- A few top products drive a significant portion of total sales

Files included
  - 'Ecommerce_sales_dashboard.pbix' - Power BI dashboard for Visualization
  - 'queries.sql'- Sql queries used for data analysis
